# Culture App

## Setup

Clone the repo and install the dependencies.

```bash
npm install
```

Create a .env file and add the following variables.

```.env
CLOUD_NAME =
API_KEY =
API_SECRET =
DB =
```

Add your mongo db url

Run the app

```bash
npm run dev
```
